# Chahel Tanna - Personal Portfolio

Welcome to my personal portfolio website repository! This project highlights my journey as a software developer, showcasing my skills, projects, and professional background.

## 🚀 Live Demo
Visit the live website here: **[chaheltanna.vercel.app](https://chaheltanna.vercel.app)**

## 📁 Repository Structure
To maintain a clean and focused repository, only the core application structure is tracked:
- `src/` - The core application source code (components, pages, styles).
- `public/` - Public assets, images, and static resources.

*(Note: Configuration files like `package.json`, `.vscode`, and `.next` are intentionally excluded from this repository).*

## 🛠️ Tech Stack
- **Framework:** [Next.js](https://nextjs.org/)
- **Library:** React
- **Styling:** Tailwind CSS
